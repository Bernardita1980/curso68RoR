@media queries

CSS uses the @media rule to detect information about the device being esed to access
the website.For example,it can be used to detect the size of de device, its capabilities,
and its orientation.For this course, we will be using it to detect the device size, but keep
in mind that it is caoable of much more.

Phone: Width less than or equal to 480px

Table: Width between 481px and 1023px

Desktop: Width greater than or equal to 1024px
